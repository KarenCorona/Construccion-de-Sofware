      6. 1. Declaración de Variables


      1.1 Explicación de las palabras clave 'let' 'const' y 'var'

      - let: Introduce una variable de alcance local en un bloque de código. Su valor puede cambiar a lo largo del tiempo.
      - const: Declara una variable de solo lectura con un valor constante. El valor de una constante no puede ser reasignado.
      - var: Declara una variable global o local en una función, sin importar el ámbito del bloque de código. Se puede reasignar y su alcance es el de la función en la que está declarada o global si está fuera de cualquier función.


      1.2 Diferencias entre 'let' 'const' y 'var'

      - let y const tienen un alcance de bloque, lo que significa que solo son accesibles dentro del bloque en el que se declaran. var tiene un alcance de función o global.
      - let y const no se ven afectados por el hoisting, lo que significa que no se pueden utilizar antes de su declaración. var, por otro lado, se eleva al principio de su ámbito y puede ser utilizada antes de su declaración.
      - const es una constante y no puede ser reasignada. let y var pueden ser reasignadas.

      2. Tipo de Datos


      2.1 Descripcion de los tipos de Variables en JavaScript

      - Number: Representa valores numéricos, enteros o de punto flotante.
      - String: Representa una secuencia de caracteres.
      - Boolean: Representa un valor lógico verdadero o falso.
      - Undefined: Representa un valor no definido.
      - Null: Representa un valor nulo o vacío.
      - Object: Representa un objeto, que puede contener propiedades y métodos.
      - Symbol: Representa un valor único e inmutable que se utiliza como clave de una propiedad de un objeto.

      2.2 Ejemplos de declaración de variables para cada tipo de dato.

      - Number: let age = 25;
      - String: let name = 'John';
      - Boolean: let isStudent = true;
      - Undefined: let x;
      - Null: let y = null;
      - Object: let person = {name: 'John', age: 25};
      - Symbol: let key = Symbol('key');

      3. Alcance de Variables


      3.1 Explicación del alcance global y local en JavaScript.

      - Global Scope: Las variables globales son accesibles desde cualquier parte del código.
      - Local Scope: Las variables locales son accesibles solo dentro de la función en la que se declaran.

      3.2 Ejemplos de variables declaradas dentro y fuera de funciones.

      - Global Scope: let x = 10;
      - Local Scope: function myFunction() { let y = 20; }

      4. Asignación de Variables


      4.1 Cómo asignar valores a variables en JavaScript.

      - Asignación directa: let x = 10;
      - Asignación con operadores: x += 5;

      4.2 Ejemplos de asignación utilizando operadores

      - Suma: let x = 10; x += 5; // x = 15
      - Resta: let y = 20; y -= 10; // y = 10
      - Multiplicación: let z = 5; z *= 2; // z = 10
      - División: let w = 10; w /= 2; // w = 5

      5. Variables en Funciones


      5.1 Uso de variables dentro de funciones.

      - Variables locales: function myFunction() { let x = 10; }
      - Variables globales: let y = 20; function myFunction() { console.log(y); }

      5.2 Ejemplos de variables locales y globales en funciones.

      - Variables locales: function myFunction() { let x = 10; console.log(x); }
      - Variables globales: let y = 20; function myFunction() { console.log(y); }


      Las variables son componentes esenciales en JavaScript y en la programación en general. Permiten a los desarrolladores almacenar y manipular datos de manera eficiente en diferentes contextos. Entender cómo declarar, asignar y utilizar variables en JavaScript es fundamental para escribir código claro y funcional. Con la información proporcionada en este artículo, esperamos que puedas aprovechar al máximo el potencial de las variables en tus proyectos de desarrollo web.
